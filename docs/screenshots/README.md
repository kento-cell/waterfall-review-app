# Screenshots

README から参照される主要画面のスクリーンショット格納先。

## 撮影対象

| ファイル名 | 画面 | 推奨撮影タイミング |
|---|---|---|
| `01_dnd_review_input.png` | レビュー実行ページ (D&D 1 画面投入) | 観点 + UI + SS をすべて投入した状態 |
| `02_review_result.png` | レビュー結果ページ | 指摘 5〜10 件 + 重要度 3 色が並んだ状態 |
| `03_finding_detail.png` (任意) | 指摘詳細ダイアログ | 対応状況プルダウン展開状態 |
| `04_pdf_overlay.png` (任意) | DL した PDF を開いた様子 | 注釈オーバーレイが分かるページ |

## 撮影手順

1. `LLM_PROVIDER=anthropic` (本物 LLM) で `samples/` を投入
2. 結果ページで指摘が複数件出てから撮影
3. PNG / 最大幅 1600px / 個人情報なし を確認
4. ファイル名を上記表に合わせて保存

## 注意

- ブラウザの URL バーに localhost 以外が映らないこと
- 開発者ツール / 拡張機能のサイドバーは閉じる
- ユーザ名 / メールは `demo@example.com` 等のサンプルに統一
